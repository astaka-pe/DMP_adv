Dual Deep Mesh Prior

One-shot Learning of Mesh Denoising with
Dually Self-supervised Graph Convolutional Networks

paper Code Slide

Abstract

This study proposed a one-shot learning framework for mesh denoising, where two graph convolutional networks that filter vertex positions and facet normals apart were trained jointly in the self-supervised manner with only a noisy input mesh. We leveraged a framework of deep image prior (DIP), a self-supervised image restoration method using a convolutional neural network (CNN), to obtain a denoised mesh without any ground-truth noise-free meshes. Compared to the original DIP that transforms a fixed random code into a noise-free image by the neural network, we reproduce vertex displacement from a fixed random code and reproduce facet normals from feature vectors that summarizes local triangle arrangements. After tuning several hyperparameters with a few validation samples, our method achieved significantly higher performance than traditional approaches working with a single noisy input mesh. Moreover, its performance is better than the other methods using deep neural networks trained with a large-scale shape dataset. The independence of our method of either large-scale datasets or ground-truth noise-free mesh will allow us to easily denoise meshes whose shapes are rarely included in the shape datasets.
